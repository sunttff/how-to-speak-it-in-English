**叫人起床**
1. 该醒来了！
2. 起床了，你这个懒鬼！
3. 醒醒，亲爱的。
   
**起床状态**
1. 我还是好想睡。
2. 我觉得昏昏沉沉的。
3. 我睡醒时眼睛又肿又有黑眼圈。
   
**贪睡、赖床、睡过头**
1. 让我再赖床十分钟。
2. 我忘了调闹钟而睡过头了。
3. 我不记得是否按了贪睡钮。

```
D：该醒来了！
N：让我再赖床十分钟啦。我还是好想睡。
D：快点！起床了，你这个懒鬼！
N：走开啦！闹钟又还没响。
D：已经响过了！你不记得你按了贪睡钮吗？
```
